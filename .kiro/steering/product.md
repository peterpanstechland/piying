# Product Overview

## 皮影互动短片生成系统 (Shadow Puppet Interactive System)

An innovative touchless interactive experience system that uses computer vision to create personalized shadow puppet videos. Users interact with the system entirely through body movements and hand gestures—no keyboard, mouse, or touch input required.

## Core Experience

Users approach the system → automatic detection → gesture-based scene selection → perform 2-4 motion segments → system generates a 30-second shadow puppet video → scan QR code to download.

## Key Features

- **Touchless Interaction**: All controls via body and hand gestures using MediaPipe
- **Motion Capture**: Records user movements as pose data across multiple segments
- **Shadow Puppet Rendering**: Transforms captured movements into artistic shadow puppet style
- **Instant Results**: Video generation completes in 10-20 seconds
- **Mobile Download**: QR code for easy video download to smartphones
- **Autonomous Operation**: Runs continuously without manual intervention, auto-resets between users

## Target Deployment

Local installation on exhibition/display computers for public interactive experiences. Designed for multi-hour continuous operation with automatic cleanup and resource management.

## Languages

Primary interface language: Simplified Chinese (简体中文)
Fallback language: English
