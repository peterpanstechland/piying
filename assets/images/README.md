## Image Specifications

### Scene Icons

**Technical Requirements:**
- **Size:** 256x256 pixels
- **Format:** PNG with transparency (RGBA)
- **Color Depth:** 32-bit
- **File Size:** < 100 KB
- **Style:** Silhouette or simplified line art

**Design Guidelines:**

**Scene A Icon (Martial Arts):**
- Central element: Kung fu figure in action pose
- Colors: Black silhouette with red/gold accents
- Style: Dynamic, powerful stance
- Suggested: Fist, kick motion, or traditional martial arts symbol

**Scene B Icon (Dance):**
- Central element: Dancing figure in graceful pose
- Colors: Purple/pink gradient or solid silhouette
- Style: Flowing, elegant movement
- Suggested: Ballet pose, flowing dress, extended arms

**Scene C Icon (Story):**
- Central element: Storytelling figure or book symbol
- Colors: Blue/orange or neutral tones
- Style: Narrative, expressive gesture
- Suggested: Open book, speech bubble, or expressive figure

### Guidance Images

**Technical Requirements:**
- **Size:** 800x600 pixels (minimum)
- **Format:** PNG or JPG
- **Color Depth:** 24-bit RGB
- **File Size:** < 500 KB
- **Style:** Clear, easy to understand

**Design Guidelines:**
- Use stick figures or simplified human forms
- Show clear pose indication
- Include arrows for movement direction
- High contrast for visibility
- Minimal background
- Simple and unambiguous

## Creating Images

See `../ASSET_CREATION_GUIDE.md` for detailed instructions on creating these images.

### Recommended Tools

**Icons:**
- Adobe Illustrator (professional)
- Inkscape (free, vector graphics)
- Figma (free, web-based)
- GIMP (free, raster graphics)

**Guidance Images:**
- Adobe Illustrator (professional)
- Inkscape (free, vector graphics)
- PowerPoint/Keynote (simple, accessible)
- Draw.io (free, web-based)

### Quick Creation Methods

**For Icons:**
1. Find or create silhouette of desired pose
2. Apply appropriate colors
3. Add simple decorative elements
4. Export as PNG with transparency at 256x256

**For Guidance Images:**
1. Use stick figure template or drawing tool
2. Position figure in desired pose
3. Add arrows for movement indication
4. Add minimal text labels if needed
5. Export as PNG at 800x600 or larger
