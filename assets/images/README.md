# Scene Images and Icons

This directory should contain:

## Scene Icons
- `sceneA_icon.png` - Icon for Martial Arts scene
- `sceneB_icon.png` - Icon for Dance scene
- `sceneC_icon.png` - Icon for Story scene

## Guidance Images
- `sceneA_segment1_guide.png` - Guidance image for Scene A, Segment 1
- `sceneA_segment2_guide.png` - Guidance image for Scene A, Segment 2
- `sceneA_segment3_guide.png` - Guidance image for Scene A, Segment 3
- (Similar for scenes B and C)

## Image Requirements

### Icons
- Size: 256x256 pixels
- Format: PNG with transparency
- Style: Simple, recognizable symbols

### Guidance Images
- Size: 800x600 pixels or larger
- Format: PNG or JPG
- Content: Clear pose examples or action descriptions
- Style: Silhouettes or stick figures work well

Place the image files in this directory before running the system.
