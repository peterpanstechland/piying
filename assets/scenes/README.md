# Scene Base Videos

This directory should contain the base videos for each scene:

- `sceneA_base.mp4` - Martial Arts Performance scene (30 seconds)
- `sceneB_base.mp4` - Dance Performance scene (30 seconds)
- `sceneC_base.mp4` - Story Performance scene (30 seconds)

## Video Requirements

- Duration: 30 seconds
- Resolution: 1920x1080 (Full HD) or 1280x720 (HD)
- Frame rate: 30 FPS
- Format: MP4 (H.264 codec)
- Content: Static or slow-moving background suitable for shadow puppet overlay

## Creating Base Videos

Base videos should provide an appropriate backdrop for the shadow puppet character. Consider:

- Traditional Chinese scenery or patterns
- Theatrical stage backgrounds
- Abstract artistic backgrounds
- Thematic elements matching the scene type

Place the video files in this directory before running the system.
