# 皮影互动系统 - 使用手册

## 🎭 系统简介

皮影互动系统是一个创新的无接触互动体验系统，使用计算机视觉技术创建个性化的皮影视频。用户通过身体动作和手势与系统交互，无需键盘、鼠标或触摸输入。

## 🚀 快速开始

### 第一次使用

1. **解压文件**
   - 将压缩包解压到 `C:\ShadowPuppet\`
   - 或任意你喜欢的位置

2. **运行安装**
   - 双击 `安装.bat`
   - 等待 2-5 分钟完成安装

3. **启动系统**
   - 双击 `启动系统.bat`
   - 浏览器会自动打开
   - 开始使用！

### 日常使用

```
启动: 双击 "启动系统.bat"
停止: 双击 "停止系统.bat"
```

## 📱 访问地址

- **用户界面**: http://localhost:8000
- **管理面板**: http://localhost:8000/admin
- **默认账号**: admin / admin123

⚠️ **重要**: 首次登录后请立即修改默认密码！

## 🎯 使用流程

### 用户体验流程

1. **靠近系统** → 自动检测到人
2. **手势选择场景** → 用手指向选择
3. **表演动作** → 跟随提示完成 2-4 个动作段落
4. **生成视频** → 系统自动生成皮影视频（10-20秒）
5. **扫码下载** → 用手机扫描二维码下载视频

### 管理员操作

1. 访问 http://localhost:8000/admin
2. 使用管理员账号登录
3. 可以进行:
   - 用户管理
   - 场景配置
   - 系统设置
   - 数据统计
   - 备份恢复

## 🔧 常见问题

### Q: 如何修改密码？

A: 
1. 登录管理面板
2. 进入"用户管理"
3. 点击"修改密码"
4. 输入当前密码和新密码

### Q: 忘记密码怎么办？

A:
1. 运行 `reset-admin-password.bat`
2. 或查看 `PASSWORD_RECOVERY_QUICK_GUIDE.md`

### Q: 摄像头无法使用？

A:
1. 检查浏览器权限（允许摄像头访问）
2. 使用 Chrome 或 Edge 浏览器
3. 检查摄像头驱动是否正常

### Q: 端口被占用？

A:
1. 运行 `停止系统.bat`
2. 或修改 `app\.env` 文件中的端口号

### Q: 如何备份数据？

A:
1. 复制 `app\data\` 整个目录
2. 或使用管理面板的"导出/导入"功能

## 📊 系统要求

### 最低配置

- Windows 10/11 (64位)
- 4核心 CPU
- 8GB 内存
- 50GB 可用磁盘空间
- 720p 摄像头

### 推荐配置

- Windows 10/11 (64位)
- 8核心 CPU
- 16GB 内存
- 100GB SSD
- 1080p 摄像头

## 🎨 展览模式

如果用于展览、博物馆等场景，可以使用全屏模式：

1. 双击 `launch-kiosk-mode.bat`
2. 系统会以全屏模式启动
3. 按 `Alt+F4` 或 `F11` 退出全屏

### 开机自启动

1. 按 `Win + R`
2. 输入 `shell:startup`
3. 将 `启动系统.bat` 快捷方式复制到此文件夹
